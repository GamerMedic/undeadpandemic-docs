Follow these steps to get **UndeadPandemic** up and running on your server:

1. **Download the plugin**  
   - From [Modrinth](https://modrinth.com/plugin/undeadpandemic)  
   - Or from [GitHub](https://github.com/GamerMedic/UndeadPandemic/releases)

2. **Install the plugin**  
   - Place the `UndeadPandemic.jar` file into your server's `/plugins` folder.

3. **Install dependencies**  
   - Download and install [Vault](https://www.spigotmc.org/resources/vault.34315/)  
   - Download an [Economy plugin](https://modrinth.com/plugins?f=categories:economy)  
   - Add both to your `/plugins` folder.

4. **Restart your server**
