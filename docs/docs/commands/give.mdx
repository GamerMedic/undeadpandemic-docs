# `/undeadpandemic give`

The `/undeadpandemic give` command allows you to give any UndeadPandemic structured item to an online player.  
This command includes **three required parameters** and **one optional parameter**.

---

## Command Structure

```plaintext
/undeadpandemic give <player> <item type> <item> [quantity]
