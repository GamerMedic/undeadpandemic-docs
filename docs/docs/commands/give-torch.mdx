# `:torch` Argument

The `:torch` argument is used with the `/undeadpandemic give` command to grant a player UndeadPandemic torches. This argument supports an optional quantity parameter.

---

## Command Structure

```plaintext
/undeadpandemic give <player> torch [quantity]
