# ğŸ„ Infected Animals

Information about infected animals in UndeadPandemic.

---

### What Happens?

When you kill an animal, there is a **percentage chance** that it will drop **Rotten Flesh** instead of the usual meat.

---

### Identifying Infected Animals

- There is **no way to check** if an animal is infected before it drops Rotten Flesh.
- You only know an animal is infected **after it drops the infected meat**.

---

### Modifying Infected Animals

- The chance for an animal to drop infected meat is **configurable** in the `config.yml` file.
