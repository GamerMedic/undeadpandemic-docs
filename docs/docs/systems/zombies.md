Everything you need to know about zombies in **UndeadPandemic**.  

The plugin allows you to add an **unlimited number of zombie types**.  

You can **enable or disable** zombie types in `config.yml`.  
If all custom zombies are disabled, the plugin will default to using **vanilla Minecraft zombies**.

To create a new type of zombie:

1. Open `config.yml`

## ğŸ§Ÿâ€â™€ï¸ Hordes

Hordes occur when large numbers of zombies spawn together in an area.  

- Currently, hordes rely on **spawn rates** and are **not directly controllable**.  
- In **v2.7.0**, hordes will be **more orchestrated** and will aggressively attack players.  
- Zombie types in hordes will spawn based on the **odds defined in `config.yml`**.  

Hordes in future versions:

- Can be timed or triggered at random  
- Players with the right permissions can start a horde using a new command  
- Will officially become a dedicated **Event** in v2.7.15
