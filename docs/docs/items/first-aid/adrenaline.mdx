# ⚡ Adrenaline

## Description

Adrenaline is a fast-acting medical item that temporarily prolongs a player’s life, giving them a chance to reach a doctor or obtain a cure for their infection.

> **Important:**  
> Adrenaline does *not* cure infection and will *not* prevent death from the disease. It only delays the inevitable.

---

## Obtaining Adrenaline

Players with the following permission can obtain Adrenaline:

- **`firstaid.give`** — Allows giving first aid items using plugin commands.

Example command:

`/undeadpandemic give <player> firstaid adrenaline [quantity]`

This command gives the specified player one or more Adrenaline units.

---

## Using Adrenaline

To use Adrenaline, a player must have:

- **`firstaid.use`** — Allows consuming first aid items.

**How to use:**

- Right-click while holding Adrenaline.

When consumed, Adrenaline immediately applies its effects, extending the player’s remaining life but not preventing death from infection.
