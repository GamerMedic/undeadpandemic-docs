## Description

A **first aid kit** is a portable bag (represented as a chest) filled with various first aid items.  
It provides a convenient way for players to carry multiple medical supplies at once.

---

## Obtaining a First Aid Kit

Players with the appropriate permission can obtain first aid kits:

- **`firstaid.give`** â€” Allows giving first aid items to players.

Example command:

`/undeadpandemic give <player> firstaid kit [quantity]`

---

## Using First Aid Kits

Using a first aid kit is simple:

1. **Place the kit on the ground.**  
2. The kit will open and **all included first aid items will drop out**.  
3. These items can then be collected and used normally.

### Important Notes

- After opening a first aid kit, the player will **receive the empty chest back**.  
- This is **intentional behaviour** â€” the empty chest can be reused for storage or decoration.
