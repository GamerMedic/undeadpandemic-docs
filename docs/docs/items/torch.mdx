## Description
Using a Torch will temporarily give the user the **Night Vision** effect for **1 second per battery life**.

Battery life is **randomly generated** when the Torch is created.

## What Does the Torch Do?
The Torch allows a player to **see in the dark** by applying the Night Vision effect for a **pre-determined amount of time**.

A Torch must have **battery life** to function.

## How Is a Torch Obtained?
A Torch can only be given using the `/undeadpandemic give` command.  
This allows server admins to control how many Torches exist in the game.

From **v3.0.0**, Torches become **craftable** using custom recipes.

## Modifying the Torch
The Torch can be modified in the `config.yml` file. This includes:
- The duration (in seconds) of the Night Vision effect per **Amp-hour (Ah)** of battery life
- The item **name**
- **Lore**
- **Material**
- **Custom Model Data**
