## Description
Rotten Flesh looks the same as the vanilla Minecraft item unless changed using a texture pack.

## What Does Rotten Flesh Do?
When eaten, Rotten Flesh causes the player to become **infected**.

## How Is Rotten Flesh Obtained?
Rotten Flesh is obtained by:
- Killing **Zombies** (as usual)
- Killing **Infected Animals**

## Modifying Rotten Flesh
Rotten Flesh can **only** be modified as a **zombie drop**.
