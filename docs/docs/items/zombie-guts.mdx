## Description
Zombie Guts can be applied to chest plates to **hide from zombies**.

## What Do Zombie Guts Do?
- Players can apply Zombie Guts to a chest plate to hide themselves from zombies.
- The effect **wears off quickly**, regardless of chest plate material.
- The effect is **instantly removed** by water, including rain.
- Players must **right-click their chest plate** with the Zombie Guts to apply it.

## How Are Zombie Guts Obtained?
- Zombie Guts can be obtained by **killing zombies**. Every zombie death produces Zombie Guts.
- The `undeadpandemic.use.zombie` permission is required to use Zombie Guts.

## Modifying Zombie Guts
- Zombie Guts **cannot be modified**.
